## 💼 核心项目

### reimagined-disco - A 股量化分析平台
一个基于 Python 和 Flask 的全功能量化交易分析系统：
- **技术架构**：Flask Web + MySQL + Redis + ECharts
- **核心功能**：
  - 主力资金追踪与抢筹分析
  - 涨停板分析与统计
  - 实时行情监控
  - 历史回测引擎
- **数据来源**：整合 Tushare、AkShare、PyWencai 多数据源
- **特色功能**：
  - 逐笔成交数据分析
  - 主力评分算法
  - 板块热点分析
  - 情绪先锋指标

### AI Agent 系统
基于 Claude Agent SDK 的智能市场分析系统：
- **stock-tuyere-analysis**：风口分析 Agent
- **stock-force-analysis**：力量分析 Agent
- **stock-xianfeng-agent**：情绪先锋 Agent

### 数据处理引擎
高性能的金融数据处理管道：
- 多数据源统一接口
- 实时数据更新
- 批量数据获取优化
- 并发控制与限流
